--==============================================
-- Learning ML Algorithm Checklist/Steps
--==============================================

1. Learn underlying maths
2. Implement Algorithm using python.
3. Learn it for Classification, Regression and Text data wise.
	3.1. Try Multiclass classification in case you are trying for classification problem.
4. Learn underfitting and overfiiting using hyperparameter tuning. (Explore some parameters which is used to solve overfiiting problem.)
5. Impact of Imbalanced Dataset. (Explore some parameters which is used to handle imbalanced dataset)
6. Impact of Outlier. (Check wether outlier can impact algorithm performance)
7. Need of feature scaling/Normalization. (Few algorithm doesn't require feature scaling)
8. Understanding time complexity of algorithm.
